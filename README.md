# YOLOv Object Detection

A real-time object detection project based on the YOLOv architecture.

## Features

- Real-time object detection
- Live camera inference
- Object tracking
- Radar visualization
- Distance estimation
- Multiple object classes (Person, Car, Motorcycle, Bottle, Phone, Cat, Dog, etc.)
- Custom trained model weights
- Easy deployment

## Technologies

- Python
- YOLOv
- OpenCV
- PyTorch
- NumPy

## Project Structure

- Source code
- Pretrained/custom weight files
- Configuration files
- Example images
- Demo videos
